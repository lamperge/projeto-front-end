Borboletas & Imagens 🦋

Nesse artigo com o objetivo de aprender a adicionar imagens, colocarei fotos de várias borboletinhas muitos bonitas.🌼

Para adicionar imagens você deve primeiramente salvar as imagens na pasta, depois colocar a tag IMG. Dentro da tag img coloque src, que significa fonte. A fonte é de onde a tag img vai buscar o arquivo, então coloque src= e entre aspas coloque o nome do arquivo completo (src="imagem.png"). Ainda dentro da tag img tem o alt que é um texto alternativo que aparece em diversas ocasiões (como quando se passa o mouse em cima de uma imagem num site por exemplo.).No alt entre as aspas, coloque o nome alternativo que você deseja para aquela imagem.
O comando final ficará:
<img src="imagem.png" alt="significado da imagem">

Se quiser colocar uma imagem que está dentro de uma sub-pasta:

<img src="sub-pasta/imagem.png" alt="significado da imagem">
Apenas coloque a subpasta dentro do source e depois uma barra com o nome do arquivo que lá consta!

Também é possível pegar imagens por meio de links.

<img src="www.linkdaimagem.com" alt="Signifcado da Imagem">
Apenas coloque o link da imagem dentro de source.
