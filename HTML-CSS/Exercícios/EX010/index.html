Imagens Dinâmicas

Para adicionar imagens dinâmicas no seu site, é necessário usar a tag <picture>.
Dentro desta tag coloque a tag <img> normalmente e adicione a imagem que você deseja para o seu site.
E dentro da tag picture acima da tag img, você adiciona a tag >source<. Dentro da tag source você vai indicar os parâmetros media, type e srcset.

Media
Em media você vai adicionar a largura do site necessária para que se troque a imagem.
Type
Em type você vai adicionar o image/ o tipo da imagem (png,jpg, etc.)
Srcset
Em srcset você vai colocar o arquivo da imagem, igualmente na tag img.
Lembrando: Você deve colocar o código da menor imagem em cima e assim sucessivamente. Dentro da tag img você vai colocar a imagem que você quer que esteja por padrão.
