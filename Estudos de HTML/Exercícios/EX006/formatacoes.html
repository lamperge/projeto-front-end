Formatações Adicionais em HTML🖊

Texto marcado

para se adicionar um texto marcado, use tag mark.

Exemplo: Lorem ipsum dolor sit amet consectetur adipisicing elit.

Vai ficar como num marca texto:

caso queira mudar a cor do marca texto utilize a folha de css style dentro da tag "mark". Ficaria assim: <mark style="background-color:blue"> Texto marcado </mark>.

Texto Excluído

Usa-se a tag "del" para a semântica de um texto que foi retirado da página. De forma literal ele adiciona uma linha no meio do texto.

Exemplo: Lorem ipsum dolor sit amet consectetur adipisicing elit. Quas praesentium enim nam, vero modi delectus.

Texto Inserido

Usa-se a tag "ins" para a semântica de um texto que foi adicionado na página. De forma literal ele traça uma linha abaixo do texto.

Exemplo:Lorem ipsum dolor sit amet consectetur adipisicing elit. Quas praesentium enim nam, vero modi delectus.

Texto Subscrito

Um texto subescrito é um texto um pouco abaixo da linha do texto, na palavra H2O por exemplo, o 2 é subscrito. Use a tag "sub".

Texto Sobrescrito

Um texto subscrito é um texto um pouco acima da linha do texto, usado em equações(x2,x3,x4) por exemplo. Use a tag "sup".

Citações

Use a tag "q" para fazer citações

Exemplo:

Às vezes, a vida vai te acertar um tijolo na cabeça. Não perca a fé. - Steve Jobs

Códigos

Para se adicionar códigos use a tag "code".

Exemplo: 

            
                class MeuPrimeiroPrograma {
            
                    public static void main() {
            
                        System.out.print("Olá, mundo");
            
                    }
            
                }
