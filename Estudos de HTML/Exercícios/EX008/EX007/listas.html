Listas OL

Para criar uma lista ordenada use a tag "ol".
Dentro tag coloque a tag "li" para cada item da lista.

Listas UL

Listas UL são listas não Ordenadas. Listas que não precisam seguir uma ordem específica. Use a tag "ul".
Também utiliza-se o li para cada item.

    Lista de Compras🛒

  • Pão
  • Canela
  • Glacê
  • Fermento
  • Café
Assim como nas listas ordenadas também podemos personalizar da mesma forma. As três opções de formatação são disc, circle e square.

Listas de Definição

Mecanismos de busca como o google dão maior atenção para sites que tenham listas de definições quando se propões a dizer o significado de algo.

Como criar uma lista de definição

  1. Use a tag "dl" para criar a lista.
  2. Para criar os itens que serão definidos use a tag "dt"
  3. E para criar a definição de cada item, use a tag "dd"
