Mídias

Aúdios

Para adicionar um audio use a tag "audio". Dentro de src coloque o arquivo de audio junto com os parâmetros:controls e autoplay

Dicas

Use também a tag source com o type, assim com nas imagens e também utilize vários tipos de arquivos diferentes, pois nem todos os navegadores são compatíveis.

Vídeos

Para incorporar vídeos do youtube ou do vimeo no seu site você precisa apenas ir no site deles, clicar em compartilhar e depois em incorporar. Eles já vão fazer o código inteiro perfeitamente pra você.
